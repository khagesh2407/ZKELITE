# Quick Start Guide for Zorabliss

## 🚀 Go Live in 2 Minutes!

### Step 1: Enable GitHub Pages (Must Do!)
1. Go to: **https://github.com/khagesh2407/ZKELITE/settings/pages**
2. Under "Build and deployment":
   - **Source**: Select `Deploy from a branch`
   - **Branch**: Select `main`
   - **Folder**: Select `/ (root)`
3. Click **Save**
4. Wait 2-3 minutes for deployment ⏳

### Step 2: Access Your Site
After deployment, your site will be live at:
```
https://khagesh2407.github.io/ZKELITE/agents/
```

### Step 3: Connect Custom Domain (Optional)
1. Go to Pages settings again
2. Under "Custom domain": Enter `zkelite.store`
3. Add DNS records to Hostinger (already done)
4. Enable HTTPS
